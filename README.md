# Определение пола крабов по морфометрическим данным с помощью машинного обучения

---

## Задачи проекта

- Загрузить и изучить датасет морфометрических данных крабов.
- Проверить данные на наличие пропусков (программно и графически).
- Проанализировать структуру и статистические характеристики признаков.
- Визуализировать распределения и взаимосвязи между выбранными параметрами.
- Построить ML-модели для автоматической классификации пола крабов.
- Сделать выводы о качестве данных и их пригодности для дальнейшего моделирования.

---

## Используемые датасеты

1. **Crabs (Крабы)**
   - Набор морфометрических данных 200 крабов двух видов и пола.
   - Используется для практики анализа признаков и визуализации распределений.
   - Классы сбалансированы, отсутствуют пропущенные значения.

---

---

## Ключевые результаты

- Проверено качество данных и отсутствие пропусков.
- Получены описательные статистики и коэффициенты корреляции.
- Построены визуализации (диаграммы, графики распределений, тепловые карты).
- Проведён анализ информативности признаков для классификации пола.
- Модели случайного леса и логистической регрессии дают точность до 97%.
- Сформулированы аналитические выводы о структуре данных и значимости признаков.

---

## Описание признаков

| Столбец | Описание |
|---------|----------|
| sp      | Вид/цветовая форма краба |
| sex     | Пол (Male/Female) |
| FL      | Длина края панциря (мм) |
| RW      | Ширина панциря (мм) |
| CL      | Средняя длина панциря (мм) |
| CW      | Средняя ширина панциря (мм) |
| BD      | Глубина панциря (мм) |

---

## Основные визуализации

- Диаграммы распределения ключевых признаков.
- Тепловые карты корреляций между параметрами.
- ROC-кривые и confusion matrix моделей.

- Наиболее информативные показатели — ширина и длина панциря.
- ML подойдёт для автоматизации биологических исследований и практики.
